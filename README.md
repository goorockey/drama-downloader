drama-downloader
=============

# Installation

    pip install -r requirements.txt

# Usage

- copy config.ini.example to config.ini, and fill it

- usage: drama_download.py [-h] [-d] [-c CONFIG]

    Drama downloader

    optional arguments:
      -h, --help  show this help message and exit
      -d          Daemon mode.
      -c CONFIG   Config file. Default is config.ini
